## 2. 레벨별 학습 전략

### 🟦 Basic Level 학습 전략

> **대상:** 토익 400-700점 목표  
> **기간:** 30일  
> **단어:** 1,200개

#### 📚 학습 목표

```
✅ 1차 목표: 기본 단어 1,200개 완전 습득
✅ 2차 목표: 토익 빈출 표현 암기
✅ 3차 목표: 예문 이해 및 활용
✅ 최종 목표: 토익 600-700점 달성
```

#### 📖 일일 학습 루틴 (45-60분)

**1단계: 신규 학습 (30분)**
```
📖 Day N 단어 40개 학습
   - 발음 확인 (5분)
   - 핵심 의미 파악 (10분)
   - 예문 3개 정독 (10분)
   - 토익 빈출 표현 암기 (5분)
```

**2단계: 즉시 복습 (15분)**
```
🔁 방금 학습한 단어 복습
   - 영어 → 한글 (5분)
   - 한글 → 영어 (5분)
   - 틀린 단어 체크 (5분)
```

**3단계: 전날 복습 (15분)**
```
📚 Day N-1 단어 복습
   - 빠른 훑어보기 (5분)
   - 예문 다시 읽기 (10분)
```

#### 🎯 주간 학습 계획

```
월요일: Day 1 (40개) + 발음 집중
화요일: Day 2 (40개) + Day 1 복습
수요일: Day 3 (40개) + Day 2 복습
목요일: Day 4 (40개) + Day 3 복습
금요일: Day 5 (40개) + Day 4 복습
토요일: Day 1-5 전체 복습 (200개)
일요일: 휴식 or 가벼운 복습
```

#### ⚠️ 주의사항

```
❌ 완벽주의 금지
   - 80% 이해되면 다음으로
   - 복습으로 채우기

❌ 진도 밀리지 않기
   - 하루라도 쉬면 습관 깨짐
   - 바쁠 땐 20개라도 하기

✅ 꾸준함이 핵심
   - 매일 같은 시간에 학습
   - 작은 성취감 쌓기
```

### 🟨 Advanced-800 Level 학습 전략

> **대상:** 토익 700-850점 목표  
> **기간:** 30일  
> **단어:** 1,200개

#### 📚 학습 목표

```
✅ 1차 목표: Basic 단어 완벽 숙지 확인
✅ 2차 목표: 중급 단어 1,200개 습득
✅ 3차 목표: 동의어/유사어 구분
✅ 최종 목표: 토익 800-850점 달성
```

#### 📖 일일 학습 루틴 (60-75분)

**1단계: Basic 복습 (15분)**
```
📚 매일 랜덤 40개 빠른 복습
   - 헷갈리는 단어 체크
   - 토익 표현 재확인
```

**2단계: 신규 학습 (35분)**
```
📖 Advanced-800 Day N (40개)
   - 단어 학습 (15분)
   - 예문 분석 (10분)
   - 동의어/반의어 정리 (10분)
```

**3단계: 실전 적용 (20분)**
```
✍️ 문제 풀이 or 작문
   - 토익 기출 유형 (10분)
   - 학습 단어로 문장 만들기 (10분)
```

#### 🎯 주간 학습 계획

```
월-금: 신규 40개 + Basic 복습 + 실전
토요일: Week 복습 + 모의고사 Part 5-6
일요일: 틀린 문제 단어 집중 복습
```

#### 💡 학습 팁

```
✅ Basic과 연결하기
   - 동의어 확장
   - 반의어 대조
   - 파생어 정리

✅ 문맥 중심 학습
   - 한 단어의 여러 의미
   - 상황별 사용법
   - 뉘앙스 차이

✅ 실전 감각 키우기
   - 토익 기출 문제
   - 실전 시간 재기
   - 오답 분석
```

---

### 🟥 Advanced-900 Level 학습 전략

> **대상:** 토익 850-990점 목표  
> **기간:** 30일  
> **단어:** 1,200개

#### 📚 학습 목표

```
✅ 1차 목표: Basic + 800 완벽 복습
✅ 2차 목표: 고급 단어 1,200개 완전 습득
✅ 3차 목표: 뉘앙스 차이 완벽 이해
✅ 최종 목표: 토익 900점+ 달성
```

#### 📖 일일 학습 루틴 (75-90분)

**1단계: 전체 복습 (20분)**
```
📚 Basic + 800 랜덤 복습
   - 매일 60개씩 순환
   - 약점 단어 집중
```

**2단계: 고급 학습 (40분)**
```
📖 Advanced-900 Day N (40개)
   - 단어 심화 학습 (15분)
   - 예문 완벽 이해 (15분)
   - 고급 표현 정리 (10분)
```

**3단계: 실전 + 작문 (30분)**
```
✍️ 종합 훈련
   - Part 5-7 문제 (15분)
   - 학습 단어로 에세이 (15분)
```

#### 🎯 주간 학습 계획

```
월-목: 신규 40개 + 종합 복습 + 실전
금요일: 주간 총정리 + 약점 보완
토요일: 실전 모의고사 전체
일요일: 오답 분석 + 단어 재정리
```

#### 💡 고급 학습 팁

```
✅ 미묘한 차이 파악
   - 격식체 vs 구어체
   - 긍정/부정 뉘앙스
   - 문맥별 선택

✅ 능동적 사용
   - 영어 일기 쓰기
   - 영어로 생각하기
   - 실제 사용 연습

✅ 원어민 표현
   - 자연스러운 표현
   - 관용구 익히기
   - 실제 사용례 분석
```

## 4. 학습 시간 관리

### ⏰ 시간대별 학습 효율

**🌅 아침 (오전 6-9시)**
```
효율: ⭐⭐⭐⭐⭐
특징: 집중력 최고, 기억력 우수
추천: 신규 단어 학습

예시:
06:00-06:30 기상 및 준비
06:30-07:00 Day N 신규 단어 30개
07:00-07:30 예문 학습 + 복습
```

**☀️ 오전 (오전 9-12시)**
```
효율: ⭐⭐⭐⭐
특징: 논리적 사고 활발
추천: 문제 풀이, 심화 학습

예시:
09:00-09:30 전날 복습
10:00-10:30 토익 문제 풀이
11:00-11:30 오답 단어 정리
```

**🌤️ 오후 (오후 1-6시)**
```
효율: ⭐⭐⭐
특징: 점심 후 졸림, 집중력 저하
추천: 가볍게 복습, 듣기 연습

예시:
13:00-13:30 점심 식사
14:00-14:30 복습 + 발음 연습
15:00-15:30 예문 읽기
```

**🌙 저녁 (오후 6-9시)**
```
효율: ⭐⭐⭐⭐
특징: 재충전 후 집중력 회복
추천: 복습, 테스트

예시:
18:00-18:30 저녁 식사
19:00-19:30 Day N 복습
20:00-20:30 자가 테스트
```

**🌃 밤 (오후 9-12시)**
```
효율: ⭐⭐⭐
특징: 하루 정리, 장기 기억 전환
추천: 최종 복습

예시:
21:00-21:30 약점 단어 집중
22:00-22:30 전체 훑어보기
23:00 취침 (충분한 수면 필수!)
```

## 5. 점수대별 목표 설정

### 🎯 TOEIC 400 → 600점

**현재 상태:**
```
- 기초 단어 부족
- 문법 기초 약함
- 듣기 이해 어려움
```

**학습 계획:**
```
📚 단어: Basic 1,200개 완벽 숙지
📖 기간: 30-45일 (천천히 확실하게)
🎯 목표: 600점 달성

세부 전략:
   1. 매일 30-40개씩 (무리하지 않기)
   2. 복습 3회 이상 필수
   3. 예문 완벽 이해
   4. 토익 기초 문법 병행
```

**예상 성과:**

## 6. 문제 해결 가이드

### 🚨 자주 겪는 문제와 해결책

#### 문제 1: "단어가 외워지지 않아요"

**원인 진단:**
```
☑️ 단어만 외우고 예문을 안 읽음
☑️ 한 번만 보고 복습 안 함
☑️ 너무 많은 양을 한 번에 학습
☑️ 집중력 부족
```

**해결 방법:**
```
✅ 예문과 함께 암기하기
   - 단어 → 예문 → 단어 순서로

✅ 복습 주기 지키기
   - 당일 3회, 다음날 2회, 3일 후 1회

✅ 학습량 조절
   - 40개가 힘들면 30개로 시작
   - 점진적으로 늘리기

✅ 학습 환경 개선
   - 조용한 공간
   - 스마트폰 알림 끄기
   - 충분한 조명
```

---

#### 문제 2: "시간이 부족해요"

**원인 진단:**
```
☑️ 계획 없이 시작
☑️ 완벽주의
☑️ 우선순위 오류
☑️ 시간 관리 부족
```

**해결 방법:**
```
✅ 핵심만 학습
   - 핵심 의미 1개
   - 예문 1개만 읽기
   - 토익 표현 1개

✅ 시간 활용
   - 출퇴근 시간 (20분)
   - 점심 시간 (15분)
   - 자기 전 (10분)

✅ 학습량 조절
   - 바쁜 날: 20개
   - 보통 날: 40개
   - 여유 날: 60개 + 복습
```

---

#### 문제 3: "금방 잊어버려요"

**원인 진단:**
```
☑️ 단기 기억에만 저장
☑️ 복습 주기 안 지킴
☑️ 수동적 학습
☑️ 수면 부족
```

**해결 방법:**
```
✅ 장기 기억 전환
   - 복습 필수 (1-3-7-14일)
   - 이미지 연상
   - 스토리 만들기

✅ 능동적 학습
   - 소리 내어 읽기
   - 직접 써보기
   - 문장 만들기

✅ 생활 습관
   - 충분한 수면 (7-8시간)
   - 규칙적인 생활
   - 스트레스 관리
```

---

#### 문제 4: "슬럼프가 왔어요"

**증상:**
```
😰 의욕 저하
😰 집중력 감소
😰 암기 효율 저하
😰 진도 나가기 싫음
```

**긴급 대응:**
```
🚨 즉시 조치 (3일간)
   - 학습량 50% 감축
   - 복습 위주로 전환
   - 압박감 줄이기

💊 회복 단계 (1주일)
   - 재미있는 컨텐츠 추가
   - 스터디 그룹 참여
   - 작은 목표 설정
   - 보상 시스템 도입

💪 재시작 (2주차)
   - 천천히 학습량 증가
   - 성취감 축적
   - 긍정적 마인드
```

**예방 방법:**
```
✅ 무리하지 않기
✅ 규칙적인 휴식
✅ 다양한 학습 방법
✅ 진행 상황 기록
✅ 작은 성취 축하
```

---

#### 문제 5: "실전에서 기억이 안 나요"

**원인:**
```
☑️ 암기만 하고 적용 안 함
☑️ 시험 긴장
☑️ 단어만 외우고 문맥 무시
```

**해결 방법:**
```
✅ 실전 연습
   - 매일 토익 문제 10개
   - 시간 재고 풀기
   - 실전과 같은 환경

✅ 적용 연습
   - 학습 단어로 문장 만들기
   - 영어 일기 쓰기
   - 말로 설명하기

✅ 긴장 완화
   - 모의고사 여러 번
   - 심호흡 연습
   - 긍정적 자기 대화
```

## 💡 성공을 위한 마지막 조언

### ✅ 성공하는 학습자의 5가지 습관

```
1️⃣ 매일 같은 시간에 학습
   → 습관 형성의 핵심

2️⃣ 복습을 절대 빼먹지 않음
   → 장기 기억의 비결

3️⃣ 작은 성취를 축하함
   → 동기 부여의 원천

4️⃣ 실수를 두려워하지 않음
   → 성장의 기회

5️⃣ 꾸준함을 최우선으로
   → 완벽함보다 중요
```

```
✅ 듣기 이해도 향상
✅ 자신감 상승

→ 더 나은 커리어
→ 더 넓은 기회
→ 더 큰 성취감
```
