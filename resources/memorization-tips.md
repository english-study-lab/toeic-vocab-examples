## 1. 기본 원칙

### 📖 문맥 속에서 암기하기

**❌ 비효율적인 방법:**
```
단어만 외우기
- accomplish = 성취하다, 달성하다
- achievement = 성취, 업적
- acquire = 얻다, 습득하다
```

**✅ 효과적인 방법:**
```
예문과 함께 암기하기
- We accomplished our goals ahead of schedule.
  (우리는 예정보다 일찍 목표를 달성했다.)
  
- His greatest achievement was building the company.
  (그의 가장 큰 업적은 회사를 세운 것이었다.)
  
- She acquired new skills through training.
  (그녀는 훈련을 통해 새로운 기술을 습득했다.)
```

**💡 왜 효과적인가?**
- 단어의 실제 사용법을 이해
- 문맥이 기억의 단서(cue)가 됨
- 토익 시험에서 바로 적용 가능

---

### 🔊 소리 내어 읽기

**방법:**
1. 발음 기호 확인 [əˈkɑːmplɪʃ]
2. 정확한 발음으로 3번 읽기
3. 예문을 소리 내어 읽기
4. 리듬과 억양에 주의

**💡 효과:**
- 청각 기억 활성화
- 발음 교정
- 더 오래 기억됨
- LC(듣기) 대비에도 도움

---

### ✍️ 손으로 직접 쓰기

**추천 방법:**
```
1단계: 단어 3번 쓰기
   accomplish
   accomplish
   accomplish

2단계: 예문 1번 쓰기
   We accomplished our goals.

3단계: 자신만의 문장 만들기
   I want to accomplish my dream.
```

**💡 효과:**
- 운동 기억(motor memory) 활용
- 집중력 향상
- 철자 정확도 향상

## 2. 암기 전략

### 🎯 전략 1: 연상 기억법

**어원 활용:**
```
accomplish
= ac(to) + com(together) + plish(fill)
= 함께 채우다 → 완성하다 → 달성하다

predict
= pre(before) + dict(say)
= 미리 말하다 → 예측하다

transport
= trans(across) + port(carry)
= 건너서 나르다 → 운송하다
```

**이미지 연상:**
```
expand (확장하다)
→ "ex(밖으로) + pand(펼치다)"
→ 부채를 펼치는 이미지 상상

collapse (무너지다)
→ "col(함께) + lapse(미끄러지다)"
→ 건물이 무너지는 이미지 상상
```

---

### 🎯 전략 2: 덩어리로 암기

**콜로케이션(연어) 암기:**
```
❌ 단어만: accomplish
✅ 덩어리로: accomplish + goal/task/mission/objective

❌ 단어만: make
✅ 덩어리로: 
   - make a decision (결정하다)
   - make a profit (이익을 내다)
   - make progress (진전을 이루다)
   - make an effort (노력하다)
```

**토익 빈출 패턴:**
```
동사 + 전치사
- rely on (~에 의존하다)
- deal with (~를 다루다)
- result in (~를 초래하다)

형용사 + 명사
- significant impact (상당한 영향)
- annual budget (연간 예산)
- prior experience (사전 경험)
```

---

### 🎯 전략 3: 그룹핑(묶음) 암기

**동의어 그룹:**
```
"감소하다" 그룹
- decrease (감소하다)
- decline (감소하다, 거절하다)
- reduce (줄이다)
- diminish (줄어들다)

"증가하다" 그룹
- increase (증가하다)
- rise (오르다)
- grow (성장하다)
- expand (확장하다)
```

**주제별 그룹:**
```
"회의" 주제
- attend a meeting (회의에 참석하다)
- schedule a meeting (회의를 일정 잡다)
- postpone a meeting (회의를 연기하다)
- chair a meeting (회의를 주재하다)
- minutes of meeting (회의록)
```

---

### 🎯 전략 4: 대조 암기

**반의어 대조:**
```
expand ↔ shrink (확장하다 ↔ 축소하다)
accept ↔ reject (받아들이다 ↔거절하다)
increase ↔ decrease (증가하다 ↔ 감소하다)
temporary ↔ permanent (일시적인 ↔ 영구적인)
```

**혼동하기 쉬운 단어:**
```
adapt (적응하다) vs adopt (채택하다)
- adapt to changes (변화에 적응하다)
- adopt a policy (정책을 채택하다)

affect (영향을 주다) vs effect (효과)
- Climate affects crops. (기후가 작물에 영향을 준다)
- The effect was immediate. (효과가 즉각적이었다)
```

## 3. 복습 시스템

### 📅 에빙하우스 망각 곡선 활용

**과학적 복습 주기:**
```
학습 당일:  100% 기억
1일 후:     58% 기억 → 1차 복습 (20분)
3일 후:     44% 기억 → 2차 복습 (15분)
7일 후:     36% 기억 → 3차 복습 (10분)
14일 후:    30% 기억 → 4차 복습 (10분)
30일 후:    25% 기억 → 최종 복습 (5분)
```

**💡 핵심:**
- 복습 간격을 점점 늘림
- 복습 시간은 점점 줄어듦
- 장기 기억으로 전환

---

### 🔄 일일 복습 루틴

**아침 (10분):**
```
📖 오늘 학습할 단어 훑어보기
   - Day N 단어 40개 빠르게 스캔
   - 아는 단어 vs 모르는 단어 구분
   - 모르는 단어에 ⭐ 표시
```

**점심 (10분):**
```
🔁 오늘 단어 1차 복습
   - 예문 소리 내어 읽기
   - ⭐ 표시 단어 집중
   - 토익 빈출 표현 암기
```

**저녁 (15분):**
```
✅ 오늘 단어 테스트 + 어제 단어 복습
   - 영어 보고 한글 뜻 말하기
   - 한글 보고 영어 단어 쓰기
   - 어제 단어 빠른 복습
```

**취침 전 (5분):**
```
💤 오늘 단어 마지막 복습
   - ⭐ 표시 단어만 집중
   - 예문 1번씩 읽기
   - 잠들기 직전 학습이 가장 효과적
```

---

### 📊 주간 복습 시스템

**주중 (월~금):**
```
매일: 신규 단어 40개 + 전날 복습
시간: 30-40분/일
```

**주말 복습:**
```
토요일: Week 1-2 전체 복습 (60분)
   - Day 1-7 또는 Day 8-14
   - 틀린 단어 집중
   - 토익 빈출 표현 재암기

일요일: 휴식 또는 가볍게 복습 (30분)
   - 헷갈리는 단어만
   - 예문 소리 내어 읽기
```

## 4. 실전 적용

### 🎯 토익 RC 적용법

**Part 5-6 (문법/어휘):**
```
1. 문맥 파악
   - 빈칸 앞뒤 읽기
   - 긍정/부정 판단

2. 단어 떠올리기
   - 학습한 단어 중 연상
   - 토익 빈출 표현 활용

3. 보기 제거
   - 품사 확인
   - 의미상 적합성 판단
```

**Part 7 (독해):**
```
1. 지문 스캔
   - 주제 파악
   - 키워드 찾기

2. 문제 확인
   - 질문 유형 파악
   - 관련 단어 확인

3. 답 찾기
   - 동의어 변환 주의
   - 학습한 표현 활용
```

---

### 🎯 토익 LC 적용법

**Part 1-2 (사진/질의응답):**
```
- 일상 동작 단어 집중
- 전치사 청취 (at, in, on)
- 유사 발음 주의
```

**Part 3-4 (대화/담화):**
```
- 비즈니스 단어 집중
- 동의어 변환 청취
- 빈출 표현 패턴 인식
```

---

### 📝 자주 틀리는 유형 대비

**동음이의어:**
```
right (오른쪽, 권리, 옳은)
- Turn right at the corner.
- You have the right to know.
- That's the right answer.
```

**다의어:**
```
address (주소, 연설하다, 다루다)
- What's your address?
- The CEO will address the audience.
- We need to address this issue.
```

**유사 철자:**
```
affect (영향을 주다) vs effect (효과, 영향)
access (접근) vs assess (평가하다)
adopt (채택하다) vs adapt (적응하다)
```

## 5. 레벨별 팁

### 🟦 Basic Level (TOEIC ~700)

**목표:**
- 기본 1,200개 단어 완벽 암기
- 토익 빈출 표현 익히기
- 예문 이해하기

**학습 방법:**
```
1단계: 소리 내어 3번 읽기
2단계: 예문 해석하며 이해
3단계: 토익 표현 암기
4단계: 다음날 반드시 복습
```

**집중 포인트:**
- ✅ 기본 의미부터 확실히
- ✅ 예문 중심 학습
- ✅ 발음 정확히
- ✅ 꾸준한 반복

---

### 🟨 Advanced-800 Level (TOEIC 800)

**목표:**
- 중급 1,200개 단어 추가
- 유사어 구분하기
- 문맥별 의미 파악

**학습 방법:**
```
1단계: Basic 단어와 연결
2단계: 동의어/반의어 함께 암기
3단계: 콜로케이션 집중
4단계: 실전 문제 풀이
```

**집중 포인트:**
- ✅ 단어 간 차이점
- ✅ 문맥별 의미
- ✅ 관용 표현
- ✅ 실전 감각

---

### 🟥 Advanced-900 Level (TOEIC 900+)

**목표:**
- 고급 1,200개 단어 완성
- 뉘앙스 차이 이해
- 원어민 수준 어휘력

**학습 방법:**
```
1단계: 미묘한 의미 차이 파악
2단계: 격식체/구어체 구분
3단계: 동의어 확장
4단계: 작문 연습
```

**집중 포인트:**
- ✅ 뉘앙스 차이
- ✅ 고급 표현
- ✅ 맥락 이해
- ✅ 능동적 사용

## 🚨 주의사항

### ❌ 피해야 할 학습법

**1. 벼락치기:**
```
❌ 시험 전날 1,200개 암기 시도
✅ 30일 꾸준히 하루 40개씩
```

**2. 수동적 학습:**
```
❌ 단어장만 보기
✅ 소리 내어 읽고 쓰기
```

**3. 완벽주의:**
```
❌ 100% 암기할 때까지 다음 안 넘어가기
✅ 80% 암기되면 다음으로, 복습으로 보완
```

**4. 단어만 암기:**
```
❌ 뜻만 외우기
✅ 예문과 함께 암기
```

**5. 불규칙한 학습:**
```
❌ 주말에 몰아서 하기
✅ 매일 30-40분씩 꾸준히
```

## 🎓 마무리

### 암기의 핵심 3원칙

```
1️⃣ 꾸준함 (Consistency)
   매일 조금씩이 가장 효과적

2️⃣ 반복 (Repetition)
   복습이 암기의 80%

3️⃣ 적용 (Application)
   실전에서 사용해야 내 것
```

### 성공 공식

```
✅ 하루 40개 × 30일 = 1,200개
✅ 1,200개 × 3레벨 = 3,600개
✅ 3,600개 = 토익 900점+ 달성 가능
```
